# Daten und Skripte zum Aufsatz "Topologien des Protests"

Dieses Repository enthält die für den Aufsatz verwendeten Daten, Skripte und begleitenden Materialien zur Analyse des serbischen Protestdiskurses anhand von Protestplakaten. Ziel war es, thematische Schwerpunkte und diskursive Tendenzen sichtbar zu machen – unter anderem durch manuelle Kategorienbildung, Topic Modeling (LDA) und eine automatisierte Kategorisierung basierend auf der Plattform [protestografija.cloud](https://protestografija.cloud).
